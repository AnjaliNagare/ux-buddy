# 🤖 UX Buddy

UX Buddy is a full-stack AI-powered tool that generates UX copy (buttons, tooltips, error messages, onboarding text) based on context and tone.

It helps designers and developers quickly create high-quality UX writing using AI.

---

## 🚀 Features

- 🎯 Generate UX copy for:
  - Buttons
  - Error messages
  - Tooltips
  - Onboarding flows
- 🎨 Tone selection:
  - Friendly
  - Professional
  - Playful
  - Minimal
- ⚡ Real-time AI responses
- 🧠 Supports:
  - Ollama (local AI)
